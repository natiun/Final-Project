    The History of CSS

    Founder

    • VHåkon Wium
    • CSS was found in 1994

    Purpose

      Since the beginning of the web's times there has been many mistakes and many great accomplishments. Trying to find a change in the web was always a job, but eventually we found (CSS) Cascading Style Sheets. The founder of CSS was VHåkon Wium in 1994. In the solution description, HTML would still be the way that websites are marked up, but CSS would become the language that described how pages should look.

    Why we use CSS

      CSS is a programming language. When we make, or write CSS, we don't tell the browser directly how to render a page. Instead, we describe the rules for HTML document swep by step and let the browsers handle the rendering. CSS follows a predictable and most importantly, a forgiving format and about anybody could pretty easily pick it up. CSS is unique because it allows for styles to cascade. The cascade means that styles can inherit and overwrite other styles that were previously declared.
